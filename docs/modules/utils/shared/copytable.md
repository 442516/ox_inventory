---
title: Utils.Copy
---

Copies contents of a table

```lua
Utils.Copy(table, deep)
```

| Argument | Data Type | Optional | Explanation                               |
| -------- | --------- | -------- | ----------------------------------------- |
| table    | table     | no       |                                           |
| deep     | boolean   | yes      | If set to true the copy will be recursive |
