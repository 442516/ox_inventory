---
title: Inventory.ClearPlayerInventory
---
Clears the players entire inventory.

!!! info
	```lua
	exports.ox_inventory:ClearInventory(inv)
	```
	
	| Argument | Type    | Optional | Explanation |
	| -------- | ------- | -------- | ----------- |
	| inv      | string / integer | no       | The id of the inventory to wipe |
